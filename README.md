## Block Content Editor for WordPress 

WordPress でブロック型の編集方法を実装するためのプラグインです。

[Sir Trevor](https://github.com/madebymany/sir-trevor-js) を利用しています。

## Feature

* ブロック型の記事編集作業
* フロントエンドでの記事編集
* ブロックの追加
